# Quiet Mode Pro Blueprints

Набор Blueprint'ов для Home Assistant для интеллектуального управления "Тихим режимом":

## Содержимое

- **trigger.yaml** — Управление запуском тихого режима, уведомления, откладывание, циклы
- **executor.yaml** — Реализация тихого режима (громкость, уведомления, свет, охрана)
- **alisa_tts.yaml** — Голосовое оповещение через Яндекс Алису
- **doorbell_template.yaml** — Шаблон обработки домофона (будущая интеграция)

## Установка

1. Скачайте Blueprint файл или используйте ссылку на GitHub:
   ```text
   https://raw.githubusercontent.com/zad1ak/ha/main/blueprint/quiet_mode/trigger.yaml
   ```
2. В Home Assistant: Настройки → Автоматизации → Blueprint → Импортировать по URL
3. Настройте необходимые поля через UI
4. Импортируйте остальные Blueprint'ы по аналогии

## Использование

- Включение тихого режима вручную или по расписанию  
- Отложенные уведомления через Android Push и Telegram  
- Циклическое повторение уведомлений до времени окончания  
- Универсальный текст сообщений с возможностью редактирования
